# 🎵 Discord Müzik Botu

YouTube ve Spotify linklerini destekleyen, 7/24 çalışan Discord müzik botu.

## Komutlar

| Komut | Açıklama |
|-------|----------|
| `!play [şarkı/link]` | Şarkı çal veya kuyruğa ekle |
| `!skip` | Şarkıyı geç |
| `!stop` | Durdur ve kuyruğu temizle |
| `!pause` | Duraklat |
| `!devam` | Devam et |
| `!queue` | Kuyruğu göster |
| `!loop` | Tekrar modunu aç/kapat |
| `!ses [1-100]` | Ses seviyesini ayarla |
| `!np` | Şu an çalanı göster |
| `!help` | Yardım menüsü |

## Railway'e Deploy

1. Bu klasörü GitHub'a yükle
2. railway.app'e git
3. "New Project" → "Deploy from GitHub repo" seç
4. Environment Variables'a `DISCORD_TOKEN` ekle
5. Deploy!
